{\footnotesize\ttfamily const double Min\+Compare\+Value = 1e-\/6}



The constant for comparing double numbers. 

If the difference between two double numbers is less than {\ttfamily Min\+Compare\+Value}, then they are assumed to be equal. The number, which is less than {\ttfamily Min\+Compare\+Value}, is assumed to be equal zero.
