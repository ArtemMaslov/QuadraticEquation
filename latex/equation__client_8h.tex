Quadratic\+Equation is the programm, which solves quadratic equations \+:)

Input rules\+: Entered numbers could contain only 6 digit after separator, else the programm prints warning. To increase digits after separator change {\ttfamily Min\+Compare\+Value} constant in {\ttfamily \mbox{\hyperlink{config_8h}{config.\+h}}}. The dot \textquotesingle{}.\textquotesingle{} and comma \textquotesingle{},\textquotesingle{} could be the decimal separator. Multiplication \textquotesingle{}$\ast$\textquotesingle{} and division \textquotesingle{}/\textquotesingle{} symbols are not allowed in the input string. Input {\ttfamily 2$\ast$x-\/1=0} is not correct, but {\ttfamily 2x-\/1=0} is correct. Each coefficient of quadratic equation can occur only once, otherwise the last one will be use in the calculatios. For example, in the input {\ttfamily 2x$^\wedge$2-\/3x+10x-\/13=0} member {\ttfamily -\/3x} will be ignored and this input qual to {\ttfamily 2x$^\wedge$2+10x-\/13=0}.
