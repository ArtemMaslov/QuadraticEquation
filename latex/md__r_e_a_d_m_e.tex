 \hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysubsection{Input rules}\label{md__r_e_a_d_m_e_autotoc_md2}

\begin{DoxyItemize}
\item The equation must be entered if the form {\ttfamily ax$^\wedge$2 + bx + c = 0}.
\item Entered numbers could contain only 6 digit after separator, else the programm prints warning. To increase digits after separator change Epsilon constant.
\item The dot {\ttfamily .} and comma {\ttfamily ,} could be the decimal separator.
\item Multiplication \textquotesingle{}$\ast$\textquotesingle{} and division \textquotesingle{}/\textquotesingle{} symbols are not allowed in the input string. Input {\ttfamily 2$\ast$x-\/1=0} is not correct, but {\ttfamily 2x-\/1=0} is correct.
\item Each coefficient of quadratic equation can occur only once, otherwise the last one will be use in the calculatios. For example, in the input {\ttfamily 2x$^\wedge$2-\/3x+10x-\/13=0} member {\ttfamily -\/3x} will be ignored and this input qual to {\ttfamily 2x$^\wedge$2+10x-\/13=0}.
