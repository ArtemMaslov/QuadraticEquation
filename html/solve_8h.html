◆ SolveTask()

Solution SolveTask ( const Coefficients *  params)

Function solves the quadratic equation.

The function gets coefficients of a quadratic equation and finds it's roots. There are no solutions: Coefficients.c == NO_SOLUTIONS. There is one solution: Coefficients.a equal to the root of the quation, Coefficients.c == ONE_SOLUTION. There are two solutions: Coefficients.a, Coefficients.b - the roots of the equation, Coefficients.c == TWO_SOLUTIONS. There are infinit number of solution: Coefficients.c == INF_SOLUTIONS

Parameters
[in]paramsCoefficients of the quadratic equation.
Returns
A structure with the roots of the equation.
